RRZE Downloads
=============

Wordpress-Plugin
----------------

Anzeige von Downloads

### Einstellungsmenü

RRZE Downloads

### Shortcode

[downloads]
------------------
Erzeugt eine Listeansicht aller in der Mediathek verfügbaren Download-Dateien.

Attribute:
<pre>
category
cat
tags
type
format
htmlpre
htmlpost
htmlitempre
htmlitempost
search_application
search_image
search_video
search_audio
search_text
showsize
showcreated
showexcerpt
showcontent
errormsg
orderby
sort
</pre>

Beispiele:
<pre>
[downloads category="folien"]
[downloads category="schulung" tags="vpn,windows,anyconnect"]
[downloads category="schulung" tags="vpn,linux,macosx" showexcerpt="true"]
</pre>
